# 🦌 No Civ, Just Deer

A cozy woodland **colony-sim with a stealth twist**. Fed up with the humans, a few deer slip into the woods to build a colony of their own — gather resources, raise buildings, keep your progress hidden from passing rangers, and grow toward the Uprising.

**▶ Play:** https://castlesauce-blip.github.io/no-civ-just-deer/

## Features
- Isometric HTML5-canvas world with autotiled shorelines and a sprite tilemap
- Animated deer (idle / walk / run / work) with job roles and a management roster
- Player-placed buildings, decorations, and bridges; interactive terrain
- Readiness ladder → tiered unlocks → the Uprising finale, then an endless sandbox
- Save/load, happiness-driven auto-forage, and sound (SFX + music)

## Controls
Drag a deer onto a glowing node to gather. Use the Build / Decorate menus to place structures. Keep suspicion low — don't let humans spot your colony.
